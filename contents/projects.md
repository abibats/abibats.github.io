### <div align="center">面向 AI for Science 的科研数据构造系统</div>
#### 项目描述：
- 构建面向 AI for Science 的科研数据构造与评测系统，覆盖从论文解析、知识抽取、图谱构建到问答样本生成与对抗质检的全流程闭环。
#### 关键实现：
- 文献解析与知识抽取：基于 MinerU + VLM 无损还原 PDF 正文 / 公式 / 表格，采用「9 维 Epistemic Unit + Gemini 抽取 + Claude 审校 + 二次修正」多模型闭环完成结构化知识抽取。
- 知识图谱构建：基于抽取单元构建含层级 DAG（17 类语义关系）、环检测与「引用 → 认知单元」语义对齐的论文级知识网络。
- 问答样本生成：实现「图谱 → 科学问答」自动化生成，串联子图采样、反向问题生成、Rubric 评分、失败反馈与改写，产出 query-rubric 训练评测集。
- 对抗评测与质检：提出 A/B 对抗样本方法（语义正确但朴素 vs 表面精致但含错），配以长度 / 引用 / Rubric Flip / 信息泄漏多维校验，检验模型对答案表面质量的过度依赖。


### <div align="center">AI Agent Harness 设计与实现</div>
#### 项目描述：
- 设计并实现一套可承载多 agent 并发执行的 AI Agent Harness，支持会话管理、工具调度、插件生命周期与工作区状态持久化。
#### 关键实现：
- 分层架构：基于 TUI ⇄ ACP ⇄ Harness ⇄ PluginHost ⇄ Workspace Storage 的分层设计，承载 ReAct loop、会话管理、工具调度、插件生命周期和工作区状态持久化。
- 多工作区并发与隔离：为每个工作区 fork 独立子进程承载插件与工具执行，经 IPC 与主进程通信，多个 agent 任务可并行、互不阻塞，单个工作区崩溃（含 OOM）被隔离在内、可独立恢复。
- 自主任务调度与容错：以全局并发信号量控制总并发、按工作区轮询队列保证公平（防止单一任务饿死其他），支持任务排队、优雅中止与异常后自动恢复。
- 工具调用安全管控：采用「串行权限校验 → 有界并发执行」两阶段模型，基于 glob 规则匹配文件 / 命令 / 网络权限，交互场景可人工确认、自主场景从严拦截高危操作。
